# test v0.1
这是开发以来的第一个释放版本
实现了基本功能
对比起windows自带的move或者复制功能
在对于有很多小文件的文件夹进行跨盘传输中具有一倍的速度（对比起直接复制粘贴）
且具有更小的磁盘占用（对比起直接压缩->传输->解压）
个人测试打包为可执行程序版本比py版本速度慢，因此建议有能力的人直接使用源码运行
